# Hostel Room Allocation and Maintenance System

**Language:** Java

## How to Run

### Using Git

```bash
git clone https://github.com/sandil1116/Hostel_Management_System.git
cd Hostel_Management_System
```

### Using Eclipse IDE

Open **Eclipse IDE**

Click **File → Open Projects from File System**

Select the cloned project folder

Ensure **JDK is properly configured**

Run the `HostelManagementSystem.java` file

---

## Project Overview

The Hostel Room Allocation and Maintenance System is an Object-Oriented Programming–based Java application designed to automate hostel room allocation and maintenance management. The system reduces manual work, minimizes errors, and provides an efficient way for administrators, students, and maintenance staff to manage hostel-related tasks.

Target Users are Hostel Administrators , Hostel Students and Maintenance Staff

## Key Features

* Automatic room allocation based on availability
* Student and room record management
* Maintenance request submission and tracking
* Report generation for occupancy and maintenance history
* Persistent data storage using file handling

OOP Concepts are applied - Encapsulation ,Inheritance,Polymorphism ,Abstraction

This project is developed for academic use .
